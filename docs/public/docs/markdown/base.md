# Base

## Typography

### Links

As easy as this:

```HTML
<a href="#">Link</a>
```

### Pre, code

`pre` and `code` are automatically formatted.

```HTML
<p>This is <code>code</code>!</p>
<pre><code>alert(1)</code></pre>
```

### Heading counters

Just add `heading-counters` class to container of all headings.

```HTML
<main class="heading-counters">
<h1>Page</h1>
  <h2>Chapter</h2>
    <h2>Section</h2>
      <h3>Part</h3>
  <h2>Chapter</h2>
</main>
```
