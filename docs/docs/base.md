## Typography

### Links

As easy as this:

```html
<a href="#">Link</a>
```
```

### Heading counters

Just add `heading-counters` class to container of all headings.

```html
<main class="heading-counters">
<h1>Page</h1>
  <h2>Chapter</h2>
    <h2>Section</h2>
      <h3>Part</h3>
  <h2>Chapter</h2>
</main>
```
